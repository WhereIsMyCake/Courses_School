# An Introduction to Causal Inference

>本章仅讨论原因的结果

## 1 因果推断的范式

假设对于一个事件，结果 $Y$ 可以分为两种可能：
 - 接受处理后的结果，记为 $Y_1$
 - 未接受处理后的结果，记为 $Y_2$

那么描述这个处理没有效果，即为 $Y_1 = Y_0$ ，效果是正向的即为 $Y_1 > Y_0$ ，效果为负向的即为 $Y_1 < Y_0$ 。


现在，我们把结果表示为
$$Y = AY_1 + (1 - A)Y_0$$
其中 $A$ 为表示处理状态的变量，取 1 表示接受处理，取 0 表示未接受处理。实际情况下我们只能观察到一种状态的结果另一种状态的结果就称为反事实 Counterfactual ，而因果推动就是一个缺失数据的问题。

## 2 理想的因果效应

我们关注群体事件经过处理后的效果，用 ACE / ATE 来衡量
$$\psi = E[Y_1 - Y_0]$$ 。

当满足条件：
 - 随机化假设 RA ，即潜在结果与是否处理分配独立
 - 一致性假设 CA ，即观测结果 $Y$ 就是对应的潜在结果

那么 ACE / ATE 可以计算为
$$\psi = E(Y_1|A=1) - E(Y_0|A=0) = E(Y|A=1) - E(Y|A=0)$$ 。

## 3 NUCA 假设与 G-formula

但是，在实际情况中，存在一些特征 $L$ 既影响是否处理事件，又影响潜在结果，这是比较处理的效果就不公平了，这种特征就是协变量。

考虑到协变量的影响，我们给出 No Unmeasured Confounding Assumption 假设，即在观察到足够多的特征 $L$ 时，在 $L$ 相同的群体内部，是否接受处理就是随机的
$$(Y_0, Y_1) \perp A|L$$ 。

现在，对于一种处理状态，我们可以先求出每个特征各自的效果，再按比例加和就是某个处理状态的效果，对于离散的 $L$ ，有
$$E(Y_a) = \sum_i E(Y|A=a, L=i)P(L=i)$$ ，

对于连续的 $L$ ，有
$$E(Y_a) = \int E(Y|A=a, L=i)f_L(i)\,di$$ 

以上就是 G-公式。

## 4 IPTW / IPW

引入协变量后，还可以通过 Inverse Probability Treatment Weighting 逆概率处理加权的办法来较为公平地计算因果效应，这个方法的本质是创建一个虚拟群体。

在给定了协变量 $L$ 的情况下，处理状态为 $A$ 的概率表示为
$$f(A|L) = P(A=a|L=i)$$ ，
那么有
$$E(Y_a) = E(\frac{I(A=a)}{f(A|L)} \cdot Y)$$ ，
计算出每一个特征下的效果再按比例平均就是总效果。

## 5 双重稳健估计 DR

双重稳健估计 Doubly Robust Estimator 结合了 G-计算和 IPTW ，提高了正确率

$$\hat{\psi_{DR}} = P_n[\frac{A}{\hat{f}(a|L)}\cdot (Y - \hat{E}(Y|a,L))+\hat{E}(Y|a,L)]$$ 。
